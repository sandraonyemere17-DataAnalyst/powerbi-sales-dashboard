# Power BI Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed to analyze three years of sales performance. It provides insights into revenue, gross profit, cost of goods sold, product returns, customer count, regional sales performance, and product trends to support data-driven decision-making.

## Objectives
- Analyze sales performance over three years.
- Track key business KPIs.
- Identify top-performing products and regions.
- Monitor customer and product trends.

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Regional Sales Performance
- Product Performance
- Interactive Slicers and Filters

## Files Included
- Power BI Dashboard (.pbix)
- Excel Dataset (.xlsx)
- Project Report (.pdf)
